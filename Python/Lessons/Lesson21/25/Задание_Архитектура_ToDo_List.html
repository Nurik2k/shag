Задание: Архитектура ToDo List

Цель: Разработать программу для управления задачами с использованием класса ToDoList.

Инструкции:

  1. Архитектура класса ToDoList:

    • Класс ToDoList создан для управления списком задач.
    • Он принимает filename в качестве аргумента конструктора, который представляет собой имя файла, в котором хранятся задачи.
    • Если файл не существует, программа должна создать пустой файл с указанным именем.

  2. Метод load_tasks:

    • Метод load_tasks загружает задачи из файла Excel, если он существует. Если файл не существует, метод просто завершает работу.

  3. Метод save_tasks:

    • Метод save_tasks сохраняет текущий список задач в файл Excel.

  4. Метод add_task:

    • Метод add_task принимает task в качестве аргумента. Он добавляет эту задачу в список.

  5. Метод view_tasks:

    • Метод view_tasks возвращает список всех задач (как даты, так и самих задач).

  6. Метод remove_task:

    • Метод remove_task принимает task в качестве аргумента. Если задача существует в списке, она удаляется, и метод возвращает True, в противном случае возвращается False.

  7. Метод clear_list:

    • Метод clear_list очищает весь список задач.

  8. Метод view_day:

    • Метод view_day принимает day в качестве аргумента (в формате "YYYY-MM-DD"). Он возвращает список задач для указанной даты.

  9. Метод search_tasks:

    • Метод search_tasks принимает keyword в качестве аргумента. Он возвращает список задач, в которых содержится указанное ключевое слово.

  10. Метод show_menu:

    • Метод show_menu отображает простое текстовое меню с возможными действиями.

  11. Метод run:

    • Метод run запускает бесконечный цикл, в котором выводится меню и ожидается ввод пользователя.
    • В зависимости от выбора пользователя, вызываются соответствующие методы.

  12. Примечание:

    • В коде используется библиотека pandas для работы с данными Excel и os и Path для проверки существования и/или создания файла.
    • Класс ToDoList должен быть готов к использованию после его инициализации.

Пример использования:

if __name__ == "__main__":
    # Создаем ToDoList и передаем имя файла
    my_todo = ToDoList("tasks.xlsx")

    # Запускаем меню
    my_todo.run()
